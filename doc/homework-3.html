  1. What is meant by long format data and wide format data? Which do we generally prefer for statistical analysis?

    Wide format data is useful when a variable has several observations, so called because it tends to have fewer rows but more columns (one per observation) than the long format, which splits each of the observations across a different row, thus resulting in more rows but fewer columns. While the wide format is used for ease of viewing, for statistical analysis, the long format is preferred.
